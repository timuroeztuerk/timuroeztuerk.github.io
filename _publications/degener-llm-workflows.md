This project develops a transparent and reproducible workflow for using large language models to turn dense historical print sources into structured research data.

The paper demonstrates the workflow by digitizing Hermann A. L. Degener's *Wer Ist's?* (1911) and discusses how historians can combine OCR, preprocessing, and LLM-assisted extraction without losing reproducibility.

Code and materials are available in the repository linked above.
